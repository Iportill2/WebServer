Especificaciones HTTP

Implementación completa del protocolo HTTP/1.1

🌐 Métodos HTTP Soportados

WebServ implementa los métodos HTTP más utilizados según RFC 7231:

GET
Solicitud y recuperación de recursos del servidor
POST
Envío de datos al servidor para procesamiento
DELETE
Eliminación de recursos específicos del servidor
